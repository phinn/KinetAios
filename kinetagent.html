不止是 AI 对话

从多引擎并发到长期记忆,从工具调用到沙箱隔离,KinetAgent 让 AI 真正为你干活。

01

多 Agent 并发

每个会话独立循环,同时跑多个任务不互相阻塞。AgentScheduler 做 DAG 拓扑调度,空闲即派任务。

02

工具调用可观测

shell / read_file / write_file / web_fetch / recall_memory,每一步工具调用都看得见。

03

跨引擎 Resume

扫描 Claude / Codex 历史 sessions,选一条续接。任务连贯,不用从头开始。

04

长期记忆

每轮对话后台抽取用户事实入库,下次自动注入。不用反复说你的偏好和上下文。

05

沙箱隔离

每个 agent 的文件操作限制在各自 workspace 内,readOnly / workspaceWrite 权限可控。

06

隐私优先

API key 走系统 Keychain,对话存本地 SQLite,不上传任何后台。你的引擎、你的 key、你的数据。
