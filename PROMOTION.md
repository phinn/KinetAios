# 🚀 KinetAios Promotion Strategy

This document outlines the promotion and marketing strategy for KinetAios.

## Brand Identity

### Tagline
**"Local-First AI Agent Dashboard"** — Emphasis on privacy, control, and cross-platform support

### Key Differentiators
1. **True Local-First** — No cloud required, SQLite for persistence
2. **Multi-Engine Support** — Flexible choice (Direct/Claude Code/Codex)
3. **Cross-Platform** — Windows 11 + macOS with native performance
4. **Power-User Focused** — Shell, git integration, long-term memory
5. **Developer-Friendly** — TypeScript, no frontend bloat, full transparency
